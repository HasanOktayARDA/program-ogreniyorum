# Terminal

## bash Configuration file

1. Open Terminal app
2. go to the home directory `cd`
3. create a file .bash_profile `code .bash_profile`
4. Add for example

```bash
alias ll="ls -la"
```

5. Clouse the terminal and restart (or `source .bash_profile`)
